# Teste para Desenvolvedor Full Stack

## Rodar o backend:

  ```bash
  cd backend
  ```

- Baixar as dependencias:



  ```bash
  npm install
  ```
  
  
- Rodar o servidor:



  ```bash
  npm start
  ```


1. Rotas:

- GET /task { application-json}
- POST /task { application-json }
- PUT /task/:id { application-json }
- DELETE /task/:id { application-json }
- GET /task/:id { application-json }
